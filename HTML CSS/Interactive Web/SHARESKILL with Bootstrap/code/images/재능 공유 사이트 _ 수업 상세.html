alert 부분
.alert 는 알림(alert) 메세지를 생성하는 클래스입니다.
.alert-warning 는 alert 클래스 뒤에서 알림 메세지의 배경색과 글자색을 결정하는 8개의 클래스 중 하나입니다.(alert-primary, alert-secondary. alert-success, alert-info, alert-warning, alert-danger, alert-light, alert-dark)
.alert-dismissible 는 알림 메세지를 닫기 위해서 link나 button 요소에 class="close"와 data-dismiss="alert"와 함께 쓰입니다.
.fade 는 알림 메세지가 사라질 때 fade 효과를 주기 위한 클래스입니다.
.show는 알림 메세지가 화면에 보이게 하는 클래스입니다.(show가 없을 경우 visiblity: hidden 과 비슷한 동작을 합니다.)
aria-* attribute(속성)들은 시각장애인을 위한 속성이라 동작과는 상관이 없습니다. 관심있는 분들은 링크를 참조해 보십시오.
&times; 는 닫는 버튼으로 문자 'x'보다 선호되는 HTML entity입니다.

Bootstrap의 grid system은 내용물을 배치하고정렬하기위해 일련의 container, row, cloumn을 사용합니다. grid system은 flexbox로 구성되어 있으면 화면 너비에 완전하게 반응합니다. 한 행(row)은 12개의 열(column)로 이루어져 있습니다.

위의 표를 참고해서 한 행에 12개의 열이 있으므로, small에서는 12개의 열이 1개로, medium에서는 12개의 열이 2개로, large에서는 12개의 열이 3개로, extra large에서는 12개의 열이 4개로 나타나게 하려면 .col-sm-12 .col-md-6, col-lg-4, col-xl-3으로 클래스를 만들면 됩니다.
나머지는 코딩의 민족 코드에서 처럼 만들면 됩니다.
